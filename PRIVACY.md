# Политика конфиденциальности

Бот — учебный тренажёр для подготовки к ISSA Inshore Skipper + SRC.

## Какие данные обрабатываются

Бот хранит **минимум** данных, необходимых для работы тренировки:

- **Ваш Telegram ID** — чтобы вести прогресс отдельно для каждого пользователя.
- **Прогресс обучения**: какие вопросы вы видели, где ошибались, статистика
  ответов, серия решённых задач T-V-M-D-C.

Данные хранятся в файлах `progress/<ваш_id>.json` на сервере бота.

## Что НЕ собирается

- ❌ Имя, телефон, переписка, контакты.
- ❌ Геолокация, платёжные данные.
- ❌ Содержимое других чатов.

Бот работает **только в личном чате** и не читает сообщения в группах.

## Хранение и удаление

- Данные хранятся, пока вы пользуетесь ботом.
- Кнопка **«♻️ Сбросить прогресс»** в меню удаляет вашу статистику.
- При перезапуске бота сохранённый прогресс не теряется; сбрасываются только
  незавершённые в этот момент экзамены/задачи.

## Передача третьим лицам

Данные **не передаются** третьим лицам и используются только для работы
тренажёра.

## Контакт

Вопросы по приватности — через владельца бота (см. профиль в Telegram).
