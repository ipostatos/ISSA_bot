# Политика конфиденциальности

Бот — учебный тренажёр для подготовки к ISSA Inshore Skipper + SRC.

## Какие данные обрабатываются

Бот хранит **минимум** данных, необходимых для работы тренировки:

- **Ваш Telegram ID** — чтобы вести прогресс отдельно для каждого пользователя.
- **Прогресс обучения**: какие вопросы вы видели, где ошибались, статистика
  ответов, серия решённых задач T-V-M-D-C.

Данные хранятся в файлах `progress/<ваш_id>.json` на сервере бота.

### Синхронизация между устройствами (если включена)

Mini App (приложение в Telegram) может синхронизировать прогресс между вашими
устройствами через серверный API. Если синхронизация активна, на сервере в
базе данных (SQLite) дополнительно хранится:

- **Telegram ID** — берётся только из подписанных Telegram данных (`initData`,
  проверка подписи HMAC-SHA256); никак иначе вас не идентифицируют.
- **Состояние обучения**: данные интервального повторения (по каким вопросам
  и когда показывать снова), дневная цель, серия, дни активности.
- **История прохождений**: краткая запись о каждом тесте — режим, сколько
  вопросов, сколько верных, процент, длительность, время. Хранятся последние
  100 записей на пользователя.

Сервер видит **только обезличенный набор**: ваш Telegram ID и учебный прогресс.
Тексты ответов, переписка, контакты на сервер не передаются. Без включённой
синхронизации приложение работает полностью на вашем устройстве (localStorage),
ничего не отправляя.

## Что НЕ собирается

- ❌ Имя, телефон, переписка, контакты.
- ❌ Геолокация, платёжные данные.
- ❌ Содержимое других чатов.

Бот работает **только в личном чате** и не читает сообщения в группах.

## Хранение и удаление

- Данные хранятся, пока вы пользуетесь ботом.
- Кнопка **«♻️ Сбросить прогресс»** в меню удаляет вашу статистику в боте.
- При перезапуске бота сохранённый прогресс не теряется; сбрасываются только
  незавершённые в этот момент экзамены/задачи.
- Серверная история синхронизации хранит последние 100 прохождений на
  пользователя (старые вытесняются автоматически). Чтобы удалить данные
  синхронизации полностью — напишите владельцу (см. ниже).

## Передача третьим лицам

Данные **не передаются** третьим лицам и используются только для работы
тренажёра.

Вопросы по приватности — через владельца бота (см. профиль в Telegram).
